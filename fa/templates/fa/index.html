General information about a user's account.

Accounts that the user is watching.

/user/{name}/watchers

Accounts that are watching the specified user.

/user/{name}/journals

Retrieves all journals posted by a user.

/user/{name}/{folder}

Gets the the first few submissions from the specified folder. Options for {folder} are gallery, scraps and favorites. By default, the first 60 submissions are returned. You can pass a parameter ?page=2 to load more.

/submission/{id}

Retrieves information about the submission with the specified id.

/journal/{id}
